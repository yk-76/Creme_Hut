# 🍰 Crème Hut Cake House Management System

A **ready-to-use bakery management system** built with **VB.NET** and **Microsoft Access**.  
Designed to manage daily operations of a cake shop, including inventory, orders, and customer records.

---

## 🖥️ Features

- Manage cakes and ingredients inventory  
- Record and track customer orders  
- Generate sales and stock reports  
- User-friendly VB.NET interface
